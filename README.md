# Shopping_online
开发实践组队大作业   
技术栈:SpringBoot Vue Mybatis

## 项目简介

系统由交易市场和后台管理审核系统两部分组成。  
后台管理审核主要是负责对于用户人员、线上发布商品的审核管理功能。交易系统主要是提供在线商品的浏览和搜索，并提供用户下单购买的主要流程;提供用户注册、填写个人信息和购买历史记录。
